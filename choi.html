초문걸의 HTML5/CSS3 학습

HTML5

HTML5는 웹 페이지의 구조를 더욱 명확하게 만들고, 다양한 기능을 추가하여 개발을 편리하게 해줍니다.

1. 제목 태그 (Heading Tags)

<h1> 태그는 가장 큰 제목을 의미하며, <h6>까지 점점 작아집니다.

2. 단락과 링크

<p> 태그는 문단을 만들며, <a href="URL"> 태그는 링크를 생성합니다.

3. ID와 앵커 링크

id 속성을 사용하여 특정 요소로 바로 이동할 수 있습니다.


<h1 id="section1">섹션 1</h1>
<a href="#section1">이동</a>

4. 텍스트 스타일링

  • <b>: 굵은 글씨
  • <i>: 기울임꼴
  • <small>: 작은 글씨
  • <sup>: 윗첨자
  • <sub>: 아랫첨자

5. 목록 태그

  • <ul>: 순서 없는 목록
  • <ol>: 순서 있는 목록
  • <li>: 목록 항목

6. 표 만들기

<table> 태그로 표를 만들고 colspan, rowspan으로 병합할 수 있습니다.

7. 미디어 태그

HTML5에서는 <img>, <audio>, <video> 태그로 멀티미디어를 쉽게 추가할 수 있습니다.

CSS3

CSS는 웹 페이지를 다양한 장치에 적응시킬 수 있습니다.

1. 변수 사용

CSS3에서는 var(--변수이름)을 사용하여 재사용 가능한 값을 저장할 수 있습니다.

2. 콘솔 출력

콘솔에서 스타일을 테스트하려면 console.log()를 사용할 수 있습니다.

3. 반응형 웹 디자인

@media를 사용하여 다양한 화면 크기에 맞게 스타일을 조정할 수 있습니다.
