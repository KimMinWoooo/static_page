김민우의 HTML5/CSS3 학습

HTML5

HTML5는 웹 페이지를 만들기 위한 최신 HTML 표준으로, 구조적이고 의미론적인 태그를 제공하여 웹 접근성을 향상시키고, 멀티미디어 기능을 강화한 버전입니다.

1. 기존 HTML vs HTML5

HTML4와 HTML5의 차이를 예제로 확인해 보겠습니다.

✅ 기존 HTML (HTML4)


<div id="header">헤더 영역</div>
<div id="nav">메뉴 영역</div>
<div id="content">본문 영역</div>
<div id="footer">푸터 영역</div>

✅ HTML5 버전


<header>헤더 영역</header>
<nav>메뉴 영역</nav>
<section>본문 영역</section>
<footer>푸터 영역</footer>

👉 HTML5에서는 의미를 가진 태그를 사용하여 코드의 가독성과 접근성을 높였습니다.

2. 멀티미디어 태그

이전 HTML에서는 플러그인을 사용해야 했던 멀티미디어 요소를 HTML5에서는 기본 태그로 지원합니다.

✅ 기존 HTML (HTML4)


<object data="video.mp4" type="video/mp4"></object>

✅ HTML5 버전


<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

👉 이제 video 태그를 사용하면 간단하게 동영상을 삽입할 수 있습니다.

캔버스와 SVG

HTML5에서는 그래픽을 직접 그릴 수 있는 canvas와 벡터 그래픽을 지원하는 svg 태그를 제공


                <canvas id="myCanvas" width="200" height="100"></canvas>
                <script>
                    var canvas = document.getElementById("myCanvas");
                    var ctx = canvas.getContext("2d");
                    ctx.fillStyle = "blue";
                    ctx.fillRect(20, 20, 150, 75);
                    </script>

CSS3

CSS3는 기존 CSS에서 향상된 기능을 제공하며, 주요 특징은 다음과 같습니다.

1. CSS3 선택자

CSS3에서는 보다 정교한 선택자를 사용할 수 있습니다.


/* 기존 CSS (CSS2) */
div p { color: blue; }

/* CSS3에서의 개선 */
div > p { color: blue; } /* 직계 자식만 선택 */
input:focus { border: 2px solid red; } /* 포커스 시 스타일 적용 */

2. 미디어 쿼리

미디어 쿼리를 사용하면 반응형 웹 디자인을 구현할 수 있습니다.


@media (max-width: 600px) {
    body { background-color: lightgray; }
}

👉 화면 크기에 따라 스타일을 동적으로 변경할 수 있습니다.

3. 애니메이션과 전환 효과

CSS3에서는 애니메이션과 전환 효과를 지원하여 더욱 부드러운 UI를 만들 수 있습니다.


/* 전환 효과 */
button {
    transition: background-color 0.5s ease-in-out;
}
button:hover {
    background-color: blue;
}

/* 애니메이션 */
@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}
.element {
    animation: fadeIn 2s ease-in;
}

👉 기존 JavaScript 없이도 애니메이션을 쉽게 적용할 수 있습니다.
